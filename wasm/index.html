sk-pqc — hybrid KEM round-trip in WASM

Pure-Rust sk-pqc core compiled to wasm32-unknown-unknown. Randomness = the browser's crypto.getRandomValues.

Honest claim: this is a hybrid scheme — secure as long as either the classical X25519 leg or the ML-KEM-768 (FIPS 203) leg holds. It is not "quantum-proof" or "unbreakable".
